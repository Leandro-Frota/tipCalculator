# Calculadora gorjetas
Projeto desafio da plataforma FrontEndMentor. Criação de uma calculadora de gorjetas com HTML, CSS e Javascript.
Foi abordado os conceitos de Eventos, Varáveis, Funções, Lógica de programação em JS. 
Pontos a destacar do projeto:
<ul>
    <li>Validação para o campo número de pessoas</li>
    <li>Funcionalidade dos botões valores dos percentuais</li>
    <li>Funcionalidade do campo percenpual personalizado (CUSTOM)</li>
    <li>Criação do botão reset</li>
</ul>
